# Porfolio
This project main goal is to expose all my projects and different experiences to any headhunter and/or HR and/or techlead that would like to know more about me.

## Features I want this project to have

| ID | Feature name | Description | Term | State |
|---|---|---|---|---|
| 1 | Home | Home page with a description of what the whole project embeds | Short term | Thinking about it |
| 2 | Interactive CV | My CV did in such a way that it would be easy to edit it | Mid Term | Thinking about it |
| 3 | PDF Generator | Generate a CV changing some HTML code into latex code | Long term | Thinking about it |
| 4 | XML Reader | Creates a CV Page based on an XML file with the appropriate format | Long term | Thinking about it |
| 5 | Personal Project | All my personnal project with a video for each that explains how this works and what are the different uses of this | Mid term | Thinking about it |
| 6 | Professional Project | All my Professional projects with more detail on them that the one I could put on a CV | Mid term | Thinking about it |
| 7 | Link the interactive CV with the other sections | Creates anchors to link then interactive CV with the other parts of the website | Mid term | Thinking about it |
| 8 | My Contacts | Based on a certain level of accreditation you could see more or less of my contact informations | Long term | Thinking about it |
